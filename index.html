Welcome to SYNTHEX

Thank you for choosing SYNTHEX — a premium Framer template for AI automation agencies. This documentation will guide you through setup, customization, and publishing.

SYNTHEX is built around a retro-futurist terminal aesthetic — dark backgrounds, neon accents, CRT effects, and monospace typography. It ships with 12 fully designed pages, 7 CMS collections, and custom components built specifically for this template.

Getting Started

2.1 Duplicate the Template

01

Open the Template in Framer Marketplace

You will see the preview and a Copy button below it.

02

Click "Copy"

04

Start Customizing

Replace placeholder content, update /Terminal/TerminalPrimary to your brand color, fill in CMS collections, then publish.

Section 03

Pages

3.1 Pages Overview

SYNTHEX has 12 pages. Static pages are directly editable on the canvas. CMS pages (with :slug) are generated automatically — you never need to build individual case, blog, or job pages manually.

3.2 Editing Content

Updating Text

01

Click any text on canvas

Single-click to select, double-click to enter editing mode.

02

Replace the content

Type your text. Changes appear instantly on canvas.

Text Styles

All typography uses global Text Styles. Updating /Heading 1 once changes every headline across the entire site.

Replacing Images

01

Click the image on canvas

Reveals image controls in the Properties panel on the right.

02

Properties panel → Replace → Upload

Best practice: export at 2× and compress to under 500KB before uploading.

Image Optimization

Large images significantly slow page load. Use WebP format when possible.

Terminal vs Brand — What's the Difference?

Terminal tokens drive the retro CRT aesthetic — code components AND buttons. In SYNTHEX the main buttons are designed as part of the terminal universe, so they use /Terminal/TerminalPrimary on hover. Brand tokens drive broader site accents like navigation states and badges. You can set them to different colors for contrast, or keep them aligned for a unified look.

Panel Size Modes

Each panel has a Size Mode that controls how it fills the layout:

ModeBehavior
FillPanel stretches to fill all available space.
FitPanel sizes to its content. Use with an optional opacity multiplier (e.g. 0.7) for partial transparency.
Panel Config Format

Each Panel field shows a summary of its config: Type, Title, SizeMode, Opacity. For example: Terminal, , Fit, 0.7 — type Terminal, no custom title, Fit mode, 70% opacity.

Layout

Supports 1–6 panels. Use the Layout property to select a preset (Auto, Stacked, Inline, Equal halves, Wide left, Wide right) or enter a custom split like 2|1 (2 panels left, 1 right).

6.2 Terminal Image

A Framer component that wraps any photo inside a retro CRT monitor frame — complete with physical casing, corner bolt details, scanline effects, and glowing accent lines. This is a design component (not a code component), built entirely in Framer.

How It Works

Photo Layer
The image displayed inside the monitor screen. Replace via Properties panel → select Photo layer → image field.
Color Tint Overlay
A semi-transparent layer using /Terminal/TerminalPrimary that tints the photo to match the terminal accent color.
Top / Bottom Labels
Four editable text labels: TopLeft, TopRight, BottomLeft, BottomRight. Double-click to edit directly on canvas.
Terminal Effects
CRT scanline and noise overlay applied at 30% opacity inside the screen area.

How to Replace the Photo

01

Click the Terminal Image component on canvas

Single-click selects the outer frame.

02

Double-click to enter the component → navigate to Photo layer

The Photo layer is inside Display → Image.

03

Properties panel → image field → Replace

Upload your photo. It will be automatically tinted and framed within the CRT monitor.

Variants

Two size variants: Regular (1060×626px) and Small (522×308px). Switch in the Properties panel.

Accent Color

The monitor glow lines and photo tint automatically use /Terminal/TerminalPrimary. Change that one token and all Terminal Image instances update across the site.

6.4 NotFoundGame (404 Page)

A fully playable retro arcade game built on canvas. Turns the 404 page into an engaging experience.

Gameplay

Phase 1: Shoot and destroy the pixel "404" letters. Phase 2: Scrolling shooter — dodge and destroy enemies to score. Reach 2000 points to win.

Controls

Desktop
Arrow keys / A / D to move. Space to shoot. Press Space on idle or game over screen to start.
Mobile
Tap anywhere to start. Hold finger and drag left/right to move. Ship shoots automatically while finger is held down.

Customization

All text strings — title, subtitle, CTA, transition message, win and game over screens — are editable in property controls. The Colors object controls accent (ship, bullets), enemy (404 digits, enemy ships), background, and text colors independently.
